# Aura — Your AI Companion

This is a web-based AI chatbot application that features a Node.js backend. It can answer general questions, solve mathematical expressions, provide Wikipedia summaries, and save your conversation history permanently using a local SQLite database.

## Features
- **Mathematics Processing:** Solves numerical expressions instantly (e.g., `what is 100 * 25`).
- **Database Storage:** Automatically stores and loads chat history using SQLite.
- **Wikipedia Lookups**: Automatically fetches summaries from Wikipedia when asked about real-world concepts.
- **Beautiful UI**: Modern, glassmorphism-inspired chat interface with dark mode and voice-to-text.

---

## Getting Started

To run this application on your local machine, you'll need to have [Node.js](https://nodejs.org/) installed. 

### 1. Clone the repository
Download or clone this project folder to your local machine.

### 2. Install dependencies
Open your terminal, navigate directly inside the project root folder, and run:
```bash
npm install
```
*This command will install the necessary packages (`express`, `sqlite3`, `cors`, and `mathjs`).*

### 3. Start the server
To boot up the backend API and database, run:
```bash
node server.js
```
You should see a message in your terminal indicating: `Server is running on http://localhost:3000`.

### 4. Open the App in your Browser
Navigate to the following address in any web browser to start using your chatbot:
[http://localhost:3000](http://localhost:3000)

## Project Structure
- `chatbot.html` — The frontend user interface structure.
- `styles.css` — Website styling and themes.
- `script.js` — Client-side logic for connecting to the API and rendering chat bubbles.
- `server.js` — Core backend infrastructure handling the REST API routing, math evaluation, and SQLite connections.
- `chat.db` — Your local database file auto-generated by SQLite (holds conversation history).

## Clearing History
To clear your chat history, simply hit the **"Clear conversation"** or **Trash** icon anywhere in the UI. This will securely erase the database on the backend and refresh the conversational state.
